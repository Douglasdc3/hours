# Hours

A minimal CLI for tracking time.

## Usage

```console
hours start blog
```

```console
hours stop
```

```console
hours status
```

```console
hours add --range '9:00-11:00' blog
```

```console
hours add --from '9:00' --to '11:00' blog
```

```console
hours cancel
```

```console
hours report --csv --from '2019-04-01' --to '2019-04-31' blog
```
